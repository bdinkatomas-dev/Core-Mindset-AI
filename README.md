# Core-Mindset-AI
Core Mindset AI je od nuly postavený LLM engine (~130M parametrů, architektura GPT-2) v PyTorchi. Implementuje špičkové prvky jako RMSNorm, SwiGLU a fused attention pro maximální výkon na GPU T4 v Google Colab. Obsahuje vlastní tokenizer a AMP FP16 tréninkovou pipeline. Ideální open-source projekt pro studium hlubokého učení.
