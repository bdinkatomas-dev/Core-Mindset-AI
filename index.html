Vize Firefly 1.5

Firefly není jen další chatovací robot. Je to "Extreme GPT" engine postavený pro uživatele, kteří vyžadují přesnost a transparentnost v myšlení stroje. Naším cílem je přinést výkon velkých modelů do rukou jednotlivců s využitím chytré optimalizace.

Pokročilé uvažování

Implementujeme dedikované sekce pro "myšlení" modelu, což umožňuje AI analyzovat problém dříve, než odpoví.

Blesková odezva

Díky architektuře Firefly 1.5 dosahujeme vysoké rychlosti generování i na běžně dostupném hardwaru.

Open Source srdce

Věříme v otevřenost. Firefly je vyvíjen s důrazem na komunitu a transparentnost kódu.

Architektura Enginu

  • Parametry: ~130 Milionů
  • Normalizace: RMSNorm
  • Aktivace: SwiGLU (SiLU)
  • Pozornost: Fused QKV Projections
  • Vrstvy: 12 L x 12 H

Tréninková Data

Model využívá formátování Reasoning-Chain:

<|user|> <|thought|> <|assistant|>

Optimalizováno pro Google Colab T4 přes AMP FP16.

Implementace v PyTorchi

Ukázka modernizovaného RMSNorm bloku z Firefly 1.5:

class RMSNorm(nn.Module): def __init__(self, dim, eps=1e-6): super().__init__() self.eps = eps self.weight = nn.Parameter(torch.ones(dim)) def _norm(self, x): return x * torch.rsqrt(x.pow(2).mean(-1, keepdim=True) + self.eps) def forward(self, x): return self._norm(x.float()).type_as(x) * self.weight
